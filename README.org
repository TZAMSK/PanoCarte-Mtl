## [IPA utilisée](https://github.com/TZAMSK/RestAPIStationnement)

Ce projet est une base pour le projet de développement d'application du cours 420-G55-RO.

Vous devez :
 - Créer une divergence par équipe
 - Ajouter tous les membres de votre équipe comme «Chargé de maintenance»
 - Remplir le [[évaluation/proposition.org][formulaire de proposition]]
 - Remplacer ce fichier README par une «page d'accueil» de votre projet
 - Facultativement, ajouter un fichier de licence

   [[./licence.png]]

RÈGLE DE FORMATTAGE: 
Variables : En style serpent, avec des underscores si le nom est composé de plus qu'un mot.
Exemple : numéro_municipal

Méthodes : En style chameau, avec une minuscule au début.
Exemple : obtenirListeStationnement()

Espaces dans les parenthèses :
Ajoutez une espace après l'ouverture et avant la fermeture.
Exemple : fun récupérerChose( val chose_id: String )

Espaces vides :
Une ligne vide après avoir nommé une classe.
Une ligne vide après une condition.
Aucune ligne vide après avoir nommé une méthode.
